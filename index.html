Belajar GIT

Pengertian GIT

Git adalah sistem kendali versi terdistribusi, yang berarti semua codebase dan riwayat kode akan tersedia di komputer setiap developer untuk memudahkan pembuatan branch dan penggabungan. Git dikembangkan oleh Linus Torvalds pada tahun 2005, dan merupakan inti atau jantung GitHub.

Perintah GIT

Menurut fungsinya, GIT memiliki berbagai perintah yang dapat digunakan:

  • git init
  • git add .
  • git commit
  • git status
  • git log
  • git checkout
  • git merge
  • git push
  • git pull
  • git clone
  • git remote
  • git fetch
  • git reset
  • git revert
  • git rm

Selain berbentuk CLI (Command Line Interface), GIT juga benrbentuk GUI (Graphical user interface) yang memudahkan untuk bekerja ketika digunakan oleh banyak programmer. Branch atau cabang. Seperti namanya, Branch dapat digunakan sebagai paralel kepada Git branch lain yang telah dibuat. Dalam proyek Git apapun, kita bisa melihat semua branch dengan menulis command berikut di command line:

  • git checkout [new_branch]

    • Untuk berpindah Branch

  • git branch -d [branch_name]

    • Untuk menghapus Branch

  • git merge [branch]

    • Untuk menggabungkan Branch

  • git branch [new_branch]

    • Untuk membuat Branch
